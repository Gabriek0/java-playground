# Java Playground ☕️

Learning Java by building simple, hands-on console applications.

## Projects
### 📚 1. Library Management System
A CLI tool to manage books, library members, and borrowing.

**Goals & Concepts Practiced:**
- **Core Java:** Object-Oriented Programming (OOP), ArrayLists, Generics, Interfaces, and Composition.
- **Architecture:** Structuring code using Entities, Repositories, Services, and Enums.
- **Utilities:** Date manipulation and handling user input.
